# Bibata Original Gruvbox Yellow

A mouse cursor theme inspired by Bibata Original (sharp edges) and Gruvbox Dark theme.

- Cursor theme: Bibata Original (normal)
- Color scheme: Gruvbox Dark

![Gruvbox Colors](/gruvbox-dark-colors.png)

XCursor sizes: `14`, `20`, `22`, `24`, `28`, `32`, `40`, `48`, `56`, `64`, `72`, `80`, `88`, `96`

- Base color: #fabd2f (yellow)
- Outline color: #282828 (black)
- Watch background color: #fabd2f (yellow)
